# Configuration

The configuration file named **config.js** had to content the token of your discord bot, and your id or the ids of all other owners.
> module.exports = {  
>ᅠᅠᅠtoken: 'BOT_TOKEN',  
>ᅠᅠᅠowner:["YOUR_DISCORD_ID"]  
>ᅠᅠ};  
ᅠᅠ
